# RedLightGreenLightYOLOv8
A custom YOLOv8 machine learning model integrated into an ASIC to alert drivers when traffic lights switch colors with the goal of reducing GHG emissions and traffic delay. 

The model has been developed using multiple methods. The main deployed code was created using OpenCV and Ultralytics YOLOv8. It is a recurrent neural network, meaning the traffic lights are detected in real time. The code was also devleoped testing the moondream software and OpenCV. Both YOLOv8 and Moondream integration function with high accuracy. The model was trained on a dataset of over 100 images, and it was preprocessed to 200 for training. The percision stood at 85 percent and recall stands at 89.
